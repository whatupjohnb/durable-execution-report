# Fonts

Drop the licensed **ABC Whyte** `.woff2` files here and uncomment the
`localFont` block in `app/fonts.ts`.

Expected files:

- `ABCWhyte-Regular.woff2`
- `ABCWhyte-Medium.woff2`
- `ABCWhyte-Bold.woff2`

Until the files are present, the site falls back to Inter for headings —
everything still builds and runs.
